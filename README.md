<h1 align="center">🎓 Computer Science & Engineering Curriculum</h1>

<p align="center">

<b>Artificial Intelligence / Machine Learning</b>.
</p>

<hr>

<h2>📖 About</h2>
<p>
Instead of following a traditional university syllabus, this repository follows a
self-designed Computer Science curriculum focused on long-term mastery.
Each semester contains only a few carefully chosen subjects, allowing every topic
to be explored through theory, implementation, projects, and problem solving.
</p>
<p>
The curriculum is intentionally designed to be sustainable alongside a full-time
university degree or professional work. Rather than maximizing the number of
subjects each semester, it prioritizes consistent progress, allowing learners to
balance their primary commitments while steadily developing expertise in
Computer Science, Competitive Programming, and AI/ML.
</p>

<hr>

<h2>🎯 Objectives</h2>
<ul>
<li>✅ Reach <b>Codeforces 2000+</b></li>
<li>✅ Master Data Structures & Algorithms</li>
<li>✅ Build strong mathematical intuition for AI</li>
<li>✅ Learn Modern C++ for high-performance programming</li>
<li>✅ Learn Python as the primary language for Machine Learning, Deep Learning, and AI development.</li>
<li>✅ Understand Machine Learning and Deep Learning from scratch</li>
<li>✅ Build production-grade AI systems</li>
</ul>
